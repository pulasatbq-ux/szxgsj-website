### 域名配置步骤
1. **登录域名注册商控制台**（如阿里云）

4. **启用HTTPS**（GitHub Pages自动提供SSL证书）

## 🚀 快速部署指南

### 步骤1：创建GitHub仓库
1. 访问 https://github.com 并登录
2. 点击右上角 "+" → "New repository"
3. 仓库名：`szxgsj-website`（或其他名称）
4. 选择 "Public"（公开）
5. 不勾选 "Initialize this repository with a README"
6. 点击 "Create repository"

### 步骤2：上传网站文件
1. 在仓库页面，点击 "uploading an existing file"
2. 将 `雄光塑胶网站` 文件夹中的所有文件拖入上传区域
3. 填写提交信息："Initial commit - 雄光塑胶官方网站"
4. 点击 "Commit changes"

### 步骤3：启用GitHub Pages
1. 进入仓库 → Settings → Pages
2. 在 "Source" 部分选择：
   - Branch: `main`
   - Folder: `/` (root)
3. 点击 "Save"
4. 等待几分钟，GitHub会生成网站URL

### 步骤4：配置自定义域名
1. 在GitHub Pages设置中，找到 "Custom domain"
2. 输入您的域名：`szxgsj.com`
3. 点击 "Save"
4. 按照提示在域名注册商处配置DNS记录

### 步骤5：验证部署
1. 访问您的GitHub Pages URL：`https://用户名.github.io/szxgsj-website`
2. 访问您的自定义域名：`https://szxgsj.com`
3. 检查所有页面功能是否正常
